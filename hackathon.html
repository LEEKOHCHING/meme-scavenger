MEME SCAVENGE AGENT

"They called it dead. I call it dormant."

1. Project Overview

MEME SCAVENGE AGENT is a Web3 loot-box experience built on BNB Smart Chain. An AI agent named Sophia — the Graveyard Whisperer — continuously collects and analyzes every meme token that has graduated on the Four.meme platform. Each token is scored by AI based on real X (Twitter) community activity. Users pay USDT and receive a curated meme token delivered directly to their wallet.

4. 100% AI-Driven Token Curation

Token curation is fully automated by AI. Every token with an X account is evaluated by xAI Grok 3, which has native real-time access to X (Twitter). A single API call searches the last 30 days of posts, assesses community health, and returns a structured score (0–100) and analysis report. No manual curation. No fixed rules.

ScoreInterpretation
86 – 100High conviction — strong narrative momentum, cultural staying power
66 – 85Active community — regular posts, real engagement, visible momentum
41 – 65Moderate pulse — some genuine community activity
11 – 40Minimal — sporadic or low-quality signals
0 – 10Silent — no meaningful posts in the last 30 days

Scores and reports are pre-generated by a batch scorer and stored in the database. When a user purchases a token, the AI report is retrieved instantly — no additional API calls at purchase time.

6. AI Score Leaderboard

To demonstrate the depth of our AI analysis, we built a public leaderboard ranking every scored Four.meme token by its Grok community score. Accessible via the ✦ HACKATHON button on the main interface.

Judges can verify: every token that has graduated on Four.meme is in our database, has been scored by Grok, and has a community activity report on record. The leaderboard shows token logo, name, contract address, score, and a report button for individual inspection. Pagination is included.

7. Business Model

A 5% platform fee is charged on every purchase, retained on-chain at the point of payment. The remaining 95% is used to execute the token swap and deliver tokens directly to the user.
